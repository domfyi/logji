Emoji's are a great way to quickly locate relevant entries in logs. This  lets you `logji.wave` instead of `console.log` for situations requiring `ðŸ‘‹emphasis`. All [compatible emojis](https://github.com/domfyi/logji/blob/master/list.js) are supported.

## Demo

```
import logji from 'logji`

logji.joy("emojis")
logji.fire("are")
logji.fire("for")   
logji.computer("programmers")
logji.cocktail("too")
```
