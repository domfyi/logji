
# Installation

```
    npm install logji
```

# Example

```
import logji from 'logji`

const sayHi = () => {
    logji.joy("emojis")           // 😂emojis
    logji.fire("are")              // 🔥are
    logji.fire("for")               // 🔥for
    logji.computer("programmers")    // 💻programmers
    logji.cocktail("too")             // 🍸too
}
```
